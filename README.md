# Movie-Recommeder

How to run?
 0)Install all dependencies (streamlit,sklearn,pandas,numpy)
 1) Download tmdb datasest from kaggle "https://www.kaggle.com/datasets/tmdb/tmdb-movie-metadata".
 2) Upload tmdb_5000_credits.csv and tmdb_5000_movies.csv into root direcotry.
 3) Run movie_recommendation_system.py by running this command in root dir terminal python movie_recommendation_system.py
 4) Then in root dir terminal run streamlit run app.py
